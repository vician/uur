## Download
```bash
wget -O /tmp/uur/telegram-purple/telegram-purple.1.3.1.tar.gz https://github.com/majn/telegram-purple/releases/download/v1.3.1/telegram-purple_1.3.1.orig.tar.gz
```
## Build
```bash
cd /tmp/uur/telegram-purple/./
cd $archivedir;
./configure;
make
```
## Install
```bash
cd /tmp/uur/telegram-purple/./
cd $archivedir;
sudo make install
```
